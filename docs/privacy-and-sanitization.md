# Privacy and Sanitization

This repository uses fictional scenarios and synthetic data.

It does not include:
- employer code
- employer prompts
- internal repositories
- screenshots
- schemas
- invoices
- credentials
- proprietary architecture
- customer data
- confidential implementation details

## Publishing checklist

Before making the repo public:

- [ ] Search for employer names
- [ ] Search for internal system names
- [ ] Search for credentials or tokens
- [ ] Confirm no screenshots are included
- [ ] Confirm no real table names are included
- [ ] Confirm no copied prompts or code are included
- [ ] Confirm all scenarios are fictional
